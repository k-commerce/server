## 😄🤓 Team
### 😄 고영훈
- UI, DB, 시스템 아키텍처 설계
- CI/CD
  - Jenkins, Docker를 이용한 빌드, 배포 자동화
  - Nginx를 이용한 Blue/Green 무중단 배포
- 테스트 코드 작성
- 카테고리, 상품, 주문 API 설계 및 구현
- 검색 기능
  - 카테고리별, 이름별, ID로 상품 검색
  - QueryDSL을 이용한 동적 쿼리 작성
- 페이징 처리
  - 무한 스크롤 방식으로 상품 10개씩 조회
  - 커서 기반 페이지네이션
### 🤓 권동욱
- UI, DB, 시스템 아키텍처 설계
- CI/CD
  - Jenkins, Docker를 이용한 빌드, 배포 자동화
  - Nginx를 이용한 Blue/Green 무중단 배포
- 테스트 코드 작성
- 회원, 배송지 API 설계 및 구현
- 인증/인가
  - Spring Security를 이용한 인증/인가
  - JWT로 Access Token, Refresh Token 구현
